# Fantasy Studio — Diffusion refinement layer (Phase 16)
# ========================================================
# Local SDXL img2img + ControlNet-depth for refining Blender procedural renders
# into photoreal/stylized outputs. Install AFTER requirements.txt:
#
#     # NVIDIA GPU (recommended) — use CUDA 12.4 wheels. Required for RTX 50-series
# (Blackwell sm_120). For older cards (RTX 30-40), cu121 also works but cu124
# is safer cross-generation.
#
#     pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
#     pip install -r requirements-diffusion.txt
#
# CPU-only (very slow, ~5 min per frame — not recommended for production):
#     pip install -r requirements-diffusion.txt
#
# After install, download the models:
#     python scripts/download_diffusion_models.py
#
# Total disk usage: ~12 GB
#   SDXL Base 1.0          ~6.6 GB
#   ControlNet-Depth SDXL  ~2.5 GB
#   VAE fp16 fix           ~0.3 GB
#   misc tokenizers        ~0.1 GB
#   torch wheels           ~2.5 GB
#
# NOTE: xformers intentionally NOT listed. PyTorch 2.2+ has built-in
# scaled_dot_product_attention which does the same job and supports newer
# GPUs (Blackwell, etc) that xformers doesn't yet.
